# Audio Assets

Place audio files here. After adding files, run `npx react-native-asset` to link them.

## Music (`music/`)

| File           | Description                                  | Format |
| -------------- | -------------------------------------------- | ------ |
| `ambient.wav`  | Chill ambient loop for menus / idle screen   | WAV    |
| `gameplay.wav` | Upbeat loop for active gameplay              | WAV    |

## SFX (`sfx/`)

| File               | Description                          | Format |
| ------------------ | ------------------------------------ | ------ |
| `swap.wav`         | Short whoosh for lane change         | WAV    |
| `near_miss.wav`    | Dramatic swoosh for near-miss        | WAV    |
| `coin.wav`         | Metallic ding for coin collect       | WAV    |
| `game_over.wav`    | Impact / explosion on collision      | WAV    |
| `countdown.wav`    | Beep for countdown steps (3, 2, 1)  | WAV    |
| `countdown_go.wav` | Higher-pitch beep for "Go!"         | WAV    |
| `shield_hit.wav`   | Shield absorbing impact              | WAV    |

## Sources

Free SFX: [jsfxr](https://sfxr.me/) for 8-bit style, [freesound.org](https://freesound.org), [pixabay.com/sound-effects](https://pixabay.com/sound-effects/).
Free music: [pixabay.com/music](https://pixabay.com/music/), [opengameart.org](https://opengameart.org).
